## 简介

Catalyst 是一个集成了多种实用功能的现代化桌面应用程序，包括系统代理管理、AI 对话、开发环境部署和统一设置管理等功能。本手册将指导您如何使用 Catalyst 的各项功能。

## 快速开始

### 安装
1. 从官方网站下载适用于您操作系统的安装包
2. 运行安装程序并按照提示完成安装

### 启动
- Windows: 从开始菜单或桌面快捷方式启动
- macOS: 从应用程序文件夹启动
- Linux: 从应用程序菜单或终端启动

## 核心功能

### 主页

主页是 Catalyst 的入口页面，展示了应用程序的核心功能和快速访问入口。

**主要特点**：
- 简洁优雅的设计风格
- 流畅的动画效果
- 快速访问核心功能的卡片
- 主题切换按钮

**使用说明**：
1. 点击功能卡片可直接跳转到相应功能页面
2. 点击右上角的主题切换按钮可在亮色和暗色主题间切换

### 系统代理

系统代理功能允许您管理和配置网络代理设置。

**主要特点**：
- 支持多种代理协议（HTTP、HTTPS、SOCKS5等）
- 可视化的代理组管理
- 实时状态监控
- 配置文件导入导出

**使用说明**：
1. 导航到"系统代理"页面
2. 点击"启动"按钮启动代理服务
3. 在代理组管理区域查看和切换代理节点
4. 在配置管理区域编辑和保存代理配置

### AI 对话

AI 对话功能集成了多种大语言模型，提供智能对话服务。

**主要特点**：
- 支持多种 LLM 提供商（OpenAI、Gemini、OpenRouter等）
- 实时对话界面
- 模型参数配置
- 对话历史管理

**使用说明**：
1. 导航到"AI 对话"页面
2. 在左侧配置面板中填写提供商、Base URL、API Key 和模型名称
3. 在右侧对话区域输入您的问题并发送
4. 系统将返回 AI 的回答

### 开发环境

开发环境功能允许您一键部署常用的开发工具和环境。

**主要特点**：
- 支持多种 IDE 和开发工具
- 一键安装功能
- 安装进度显示
- 已安装软件列表

**使用说明**：
1. 导航到"开发环境"页面
2. 在工具列表中找到需要安装的软件
3. 点击"安装"按钮开始安装
4. 等待安装完成

### 设置

设置功能提供了统一的配置管理界面。

**主要特点**：
- 分类设置管理（用户、通用、LLM、代理、备份）
- 用户个性化设置
- 配置导入导出
- 全局浮窗提醒

**使用说明**：
1. 导航到"设置"页面
2. 使用顶部标签页切换不同类别的设置
3. 修改相应设置后点击"保存设置"按钮
4. 使用"配置管理"区域导出、导入或重置配置

## 高级功能

### 主题切换
Catalyst 支持亮色和暗色两种主题模式：
- 点击主页或设置页面的太阳/月亮图标切换主题
- 支持跟随系统主题自动切换

### 配置同步
- 使用"设置"页面的配置管理功能导出和导入配置
- 支持配置备份和恢复

## 故障排除

### 代理无法启动
1. 检查端口是否被其他程序占用
2. 确认配置文件格式正确
3. 查看日志获取更多信息

### AI 对话无响应
1. 检查网络连接
2. 确认 API 密钥正确
3. 查看提供商服务状态

### 安装工具失败
1. 检查网络连接
2. 确认系统权限足够
3. 查看安装日志获取详细信息

## 常见问题

### 如何获取 API 密钥？
您需要从对应的 LLM 提供商获取 API 密钥：
- OpenAI: https://platform.openai.com/
- Gemini: https://ai.google.dev/
- OpenRouter: https://openrouter.ai/

### 如何配置系统代理？
当前版本需要手动设置系统代理：
- HTTP 代理: 127.0.0.1:7890
- HTTPS 代理: 127.0.0.1:7890
- SOCKS 代理: 127.0.0.1:7891

未来版本将支持自动设置系统代理。

### 如何备份配置？
1. 导航到"设置"页面
2. 在"备份"标签页中点击"导出配置"
3. 选择保存位置并保存配置文件

### 如何恢复配置？
1. 导航到"设置"页面
2. 在"备份"标签页中点击"导入配置"
3. 选择备份文件并确认恢复
